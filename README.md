# Digital Marketing Campaign Analysis

## Objectif
Analyser la performance de 300 000 campagnes marketing sur 4 canaux (Instagram, Facebook, Twitter, Pinterest) pour optimiser le ROI.

## Stack technique
- **Python / Pandas** — nettoyage et feature engineering
- **MySQL / DBeaver** — analyse SQL
- **Matplotlib / Seaborn** — visualisations
- **Power BI** — dashboard interactif

## Dataset
- Source : Kaggle Social Media Advertising Dataset
- Volume : 300 000 lignes, 16 colonnes

## Insights clés
- ROI moyen global : **3.18** sur les 4 canaux
- **Pinterest** sous-performe : revenue moyen 97k$ vs 165k$ pour les autres canaux
- Budget total analysé : **2.3 milliards$**
- CTR moyen : 31.42% (dataset synthétique)
